# For Authors Page: Design & Data Visualization Specifications

## Page Architecture

### Overall Layout Philosophy
- **Reading width:** Max 800px for body content (literary readability)
- **Visual breaks:** Every 2-3 paragraphs, introduce data visualization or visual element
- **Hierarchy:** Clear H1 > H2 > H3 structure with ample whitespace
- **Trust signals:** Data sources cited subtly; professional, academic tone

## Color Usage Guide for This Page

**Primary Color (Ink Blue #1e3a8a):**
- CTAs
- Data chart accents
- Hyperlinks
- Icons

**Accent Color (Margin Red #dc2626):**
- ROI highlights
- Important statistics
- Comparison table negatives

**Background Colors:**
- White (#ffffff): Main content areas
- Parchment (#faf8f5): Alternating sections, cards
- Literary Black (#0a0a0a): Hero section background (subtle)

**Text Colors:**
- Literary Black (#0a0a0a): Body text, headers
- 70% Literary Black: Secondary text
- 50% Literary Black: Micro-copy, citations

## Typography Scale for This Page

**Playfair Display (Serif - Display):**
- H1: 48pt / 56pt (desktop / mobile)
- Large numbers in stat cards: 48pt
- Pricing: 36pt

**Crimson Pro (Serif - Body):**
- H2: 36pt / 42pt
- H3: 24pt / 28pt
- Pull quotes: 20pt
- Table headers: 16pt

**Inter (Sans-serif - Interface):**
- Body text: 16pt
- Captions: 14pt
- Micro-copy: 12pt
- Buttons: 16pt, medium weight

**Line Heights:**
- Headlines: 1.2
- Body text: 1.6
- Captions: 1.4

## Responsive Behavior

**Desktop (1200px+):**
- Full three-column layouts for stat cards, pricing
- Side-by-side comparison tables
- Expanded data visualizations

**Tablet (768px - 1199px):**
- Two-column layouts
- Stacked pricing cards
- Condensed comparison tables

**Mobile (< 768px):**
- Single column throughout
- Horizontal scroll for comparison tables (if needed)
- Vertically stacked pricing cards
- Touch-friendly accordions for FAQs

---

## Interactive Elements

**Hover States:**
- Cards: Subtle lift (2px) + shadow
- Buttons: Darken by 10%
- Links: Underline appears

**Scroll Animations (Optional):**
- Stat cards count up on scroll into view
- Process flowchart reveals left-to-right
- Charts draw/animate on appearance

**Sticky Elements:**
- CTA button (fixed at bottom on mobile)
- Progress indicator showing page sections

## Development Notes

**Performance:**
- Lazy load images below fold
- Inline critical CSS for above-fold content
- Optimize all chart graphics as SVG where possible

**Analytics Tracking:**
- Track scroll depth (25%, 50%, 75%, 100%)
- Button clicks on all CTAs
- FAQ accordion expansions
- Time spent on page

**A/B Testing Opportunities:**
- Hero CTA copy variations
- Pricing presentation (monthly vs. per-review)
- Testimonial vs. data-first opening

---

## Content Hierarchy Priority

1. **Attention:** Data-driven headline + immediate value prop
2. **Interest:** Powerful statistics (270%, 88% correlation)
3. **Desire:** Clear process + ethical differentiation
4. **Action:** Multiple CTAs at strategic points

**Key Principle:** Let data tell the story, but keep the human element present through testimonials and genuine reader connection.
